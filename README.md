## Overview
This JavaScript Quiz App provides an engaging interface for users to answer multiple-choice questions, receive instant feedback, and view their final score. It’s perfect for learning JavaScript DOM manipulation and working with event listeners.



## Why This Project Matters
- Enhances JavaScript skills: DOM manipulation, arrays, and event handling.  
- Strengthens logic-building and conditional reasoning.  
- A fun project to showcase in your portfolio.  



## Features
- **Multiple-choice questions** – Fully dynamic and easily expandable.  
- **Score tracking** – Tracks correct answers and displays results at the end.  
- **Instant feedback** – Shows right/wrong immediately after each answer.  
- **Restart option** – Play the quiz multiple times.  
- **Responsive design** – Works seamlessly on all screen sizes.  



## Technologies Used
- HTML5 – Structuring the content  
- CSS3 – Styling the app  
- JavaScript (ES6) – Quiz logic and dynamic updates
